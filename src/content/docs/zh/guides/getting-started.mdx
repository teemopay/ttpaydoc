## 简介

Teemopay 是一个全球化的支付平台，为全球的商户提供支付解决方案。
我们的目标是为商户提供一个简单、安全、高效的支付解决方案，帮助商户快速接入全球市场。

在实现愿景的第一站，我们选择进入南美市场，随着业务的发展，我们将逐步拓展，
感谢您的支持与陪伴，我们将不断努力，为您提供更好的服务。

## 步骤

<Steps>

1. 商务对接

   a. 确认您的价格和结算周期；

   b. 我们为您开通测试商户和对应国家的账户；

2. 阅读接入指南

   a. 阅读 [操作手册](/zh/guides/operation-manual) 了解商户后台操作；

   b. 阅读 [创建密钥](/zh/guides/create-keys) 了解如何创建和使用密钥；

   c. 阅读 [鉴权示例](/zh/guides/authentication) 了解如何进行鉴权；

3. 创建密钥

   a. 依照 [使用密钥](/zh/guides/create-keys) 创建您的公私钥；

   b. 在 Teemopay 商户后台，将您的公钥与平台公钥进行交换；

4. 接入支付

   a. 根据您的需求阅读对应国家的 API 文档；

   b. 根据 API 文档结合您的需求进行开发；

   c. 将您的开发周期和上线计划与我们分享，以便我们为您提供支持；

5. 发布上线

   a. 发布上线前，我们为您开通生产环境；

   b. 少量灰度后，正式上线；

</Steps>
